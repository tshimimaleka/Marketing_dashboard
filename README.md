
# Marketing Performance Dashboard – Power BI

Project Overview

This project presents a Marketing Performance Dashboard built in Power BI to evaluate advertising efficiency across multiple channels.

The goal of this analysis was to:

Measure Return on Ad Spend (ROAS)

Track campaign performance

Analyse conversion efficiency

Identify top-performing channels and regions

Evaluate device-based revenue contribution

# Key KPIs

Total Campaigns: 581

Total Revenue: 3.71M

Total Spend: 475.77K

Total Conversions: 46K

Conversion Rate: 7.53%

ROAS: 7.79x

CPA: 10.30

# Dashboard Features

Revenue by Channel
Spend by Channel
Revenue by Device
Revenue by Region
Revenue by Discount Usage
Day Performance Trends

# Business Insights

Strong ROAS of 7.79 indicates high advertising efficiency.

Desktop generates the highest revenue contribution.

Johannesburg is the top-performing region.

TikTok has the highest spend allocation.
